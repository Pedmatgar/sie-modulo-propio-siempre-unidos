# SIE - Módulo Propio Siempre Unidos

Módulo personalizado de Odoo para la gestión de miembros del sistema SIE (Siempre Unidos).

## Descripción

Este módulo proporciona funcionalidades para la gestión y seguimiento de miembros dentro del sistema SIE, incluyendo:

- Registro de miembros con código único autogenerado
- Gestión de estados (Borrador → Activo → Inactivo)
- Búsqueda y filtrado avanzado
- Campos de contacto (correo electrónico, teléfono)

## Instalación

1. Copia el módulo en el directorio `addons` de tu instalación de Odoo.
2. Actualiza la lista de aplicaciones desde el menú de configuración.
3. Busca **SIE - Módulo Propio Siempre Unidos** e instala.

## Uso

Una vez instalado, aparecerá el menú **SIE** en la barra de navegación principal con las siguientes opciones:

- **Miembros**: Gestiona los miembros registrados en el sistema.

## Compatibilidad

- Odoo 16.0
